API tokens for scanner ingest. The kubeposture-import CronJob on each target cluster uses a token to POST Trivy/Kyverno CRDs to /api/v1/ingest/.

How to add a cluster: create a token here, copy it, paste it into the cluster's values.yaml under token.value (or create a Kubernetes secret and reference it), then run helm install kubeposture-import. The cluster auto-registers on first ingest.

Rotate: click Regenerate to invalidate the old token and issue a new one. Revoke: click Delete to remove the service account entirely. The token value is only shown once — copy it immediately.

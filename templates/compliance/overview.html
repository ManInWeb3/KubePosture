{% help "Trivy Frameworks" %}

Each card is one industry framework (CIS Kubernetes Benchmark, NSA Hardening, Pod Security Standards, SOC2, PCI DSS…). The pass rate is an aggregate across all clusters with data for that framework.

How data arrives: Trivy Operator runs ClusterComplianceReport scans on each target cluster; the webhook posts results here. Frameworks themselves are loaded from fixture YAMLs via manage.py load_frameworks — add more by editing frameworkFixtures in the Helm values.

How to use: click View Matrix on a card to see per-control PASS/FAIL across every cluster, with remediation guidance and links to the underlying findings.
